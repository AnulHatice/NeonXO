# NeonXO - .NET MAUI Tic Tac Toe

.NET MAUI ile geliştirdiğim, neon temalı bir Tic Tac Toe (XOX) oyunu. Hem mobil hem de masaüstü platformlarda çalışacak şekilde kurguladım.

## Öne Çıkanlar

* **Tasarım:** Karanlık mod tabanlı neon arayüzü.
* **Platform:** .NET MAUI sayesinde tek kod tabanı ile iOS ve macOS/Windows desteği.
* **Arka Plan:** C# ile yazılmış, kazanma ve beraberlik durumlarını kontrol eden oyun algoritması.
* **Temiz Repo:** Derleme klasörleri (`bin/obj`) ve IDE kalıntıları temizlenmiş sade proje yapısı.

## Teknolojiler

* C# (.NET 9.0)
* XAML
* .NET MAUI
